# Word-Counter
编程项目描述：

实现一个命令行文本计数统计程序。能正确统计导入的纯英文txt文本中的字符数，单词数，句子数。
 
具体命令行界面要求如下：
 
命令模式： wc.exe [参数] [文件名]
 
例：wc.exe -c file.txt 统计字符数
      wc.exe -w file.txt 统计单词数
 
程序和参数名可以按个人喜好命名

扩展功能*（加分项）：统计代码行、空行、注释行*等，并提供相应命令接口。
